

给一棵二叉树，找出从根节点到叶子节点的所有路径   
 
样例  
样例 1:  

输入：{1,2,3,#,5}  
输出：["1->2->5","1->3"]  
解释：  
   1  
 /   \  
2     3  
 \  
  5  
样例 2:  
  
输入：{1,2}  
输出：["1->2"]  
解释：  
   1  
 /     
2
